WLEDRemote+ Web Installer

Flash the latest WLEDRemote+ firmware directly from your browser. This installer is intended for the LilyGo T-Display S3 version of the project and uses ESP Web Tools for USB flashing.

Browser flashing No esptool needed USB data cable required

Install firmware

If the button does not work, use a Chromium-based browser such as Chrome, Edge, or Brave and connect the device with a USB data cable.

Before you start

  • Use a USB cable that supports data, not only charging.
  • Close serial monitors or other tools that may already use the device.
  • For a first install, erase and flash the device when prompted.
  • After flashing, the board may reboot automatically; if not, press reset once.

Setup after flashing

  1. Boot the device and join the Wi-Fi setup flow if no saved credentials exist.
  2. Configure or discover your WLED instance.
  3. Open the local config page on port 8080 if needed.
  4. Pair it with Apple Home from the on-device HomeKit UI flow.

Troubleshooting

  • If no serial port appears, reconnect the cable and confirm browser USB permissions.
  • If flashing succeeds but the board does not boot, retry with a full erase first.
  • If the installer page is hosted on GitHub Pages, make sure manifest.json and all referenced .bin files are in the same published folder.

Release assets for this installer should include bootloader.bin, partitions.bin, boot_app0.bin, and firmware.bin.
